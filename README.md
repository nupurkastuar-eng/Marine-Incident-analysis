## Overview
This project analyzes marine incident reports from the UK Government to identify trends and patterns related to maritime safety.  
Using web scraping, NLP, and topic modeling, it extracts insights that can help improve operational safety and compliance in maritime logistics.

## Tools & Technologies
- **Languages:** Python  
- **Libraries:** Selenium, Pandas, NumPy, NLTK, SpaCy, BERT, Matplotlib, Seaborn, WordCloud  
- **Visualization:** Charts, word clouds, and topic modeling results  
- **Workflow Tools:** Jupyter Notebook / Exported HTML
